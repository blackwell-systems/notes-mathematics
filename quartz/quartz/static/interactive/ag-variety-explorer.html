Affine Variety Explorer: smooth curves, nodes & cusps

Plots the real zero set { (x, y) : f(x, y)=0 } of a plane curve and marks its singular points — the places where f = ∂f/∂x = ∂f/∂y = 0 all vanish at once.

A point is singular when both partial derivatives vanish there on the curve: the gradient gives no well-defined tangent line, so the variety is not locally a smooth curve (a node has two crossing branches, a cusp a single sharp point).
