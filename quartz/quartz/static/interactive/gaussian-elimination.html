Gaussian Elimination Stepper

Enter an augmented matrix [A | b] (read "A bar b"). The stepper runs forward elimination to row echelon form, then reduces to RREF, one elementary row operation at a time. All arithmetic is exact (rational), so the final classification is reliable.

Size: Presets:
pivot (the entry we eliminate with) entry being cleared this step

A system is unique when every variable has a pivot; inconsistent when a row reduces to [0 … 0 | k] with k ≠ 0 (an impossible 0 = k); and has infinitely many solutions when a variable has no pivot (a free variable) but the system stays consistent. The number of pivots is the rank.
