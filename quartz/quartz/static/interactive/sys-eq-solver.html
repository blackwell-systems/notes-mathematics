Line 1: a₁x + b₁y = c₁
Line 2: a₂x + b₂y = c₂
Solution (intersection)

Cramer's rule: with D = a₁b₂ − a₂b₁, the solution is x = (c₁b₂ − c₂b₁)/D and y = (a₁c₂ − a₂c₁)/D. When D = 0 the lines are parallel (no solution) or identical (infinitely many).
