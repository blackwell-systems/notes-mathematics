Relation Properties Checker

Toggle the matrix to build a binary relation R on {1,…,n}. Cell (row i, col j) on means (i, j) ∈ R.

Rows = first element, columns = second.

Directed graph of R (curved arrows; loops = reflexive pairs).

A relation is an equivalence relation when it is reflexive, symmetric, and transitive, and a partial order when it is reflexive, antisymmetric, and transitive.
