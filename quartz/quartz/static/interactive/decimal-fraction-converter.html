Decimal ↔ Fraction Converter

Every rational number has a decimal that either terminates or repeats, and every terminating-or-repeating decimal is a fraction: this tool shows the exact arithmetic both ways.

Integers only, q ≠ 0. Result updates as you type.
Examples: 1/3 2/7 1/6 22/7 1/8
For 0.1(6): whole=0, non-repeating=1, repeating=6. Leave repeating empty for a terminating decimal.
Examples: 0.(3) 0.(27) 0.1(6) 0.(142857) 0.(9)
