Newton's method fits a tangent line at the current point and jumps to where that line crosses the x-axis; near a simple root it converges quadratically (the number of correct digits roughly doubles each step), but a poorly chosen start can diverge or cycle forever, as with f(x)=x³−2x+2 from x0=0.
