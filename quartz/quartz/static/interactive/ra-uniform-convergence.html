Uniform vs. pointwise convergence of fn(x)

Pick a function sequence and slide n. We plot fn (blue) against its pointwise limit f (dashed gray), and measure the worst-case gap ∥fn − f∥∞ = supx |fn(x) − f(x)| (red segment).

fn(x) pointwise limit f(x) location of max gap
n = 3
Uniform: the entire graph of fn fits inside an ε-tube around f (one ε works for all x).   Pointwise: each x eventually gets close to f(x), but maybe not all at the same rate, so the tube never closes.

The sup-norm gap is the single number that decides it: if it marches to 0 the convergence is uniform; if it stays pinned above 0 the limit is only pointwise.
