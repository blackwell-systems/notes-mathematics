Asymmetry:

Decomposition: DKL(P||Q) = H(P,Q) − H(P)  (cross-entropy minus entropy)

Two key properties. (1) Non-negativity: DKL ≥ 0 always, and equals 0 iff P = Q. (2) Asymmetry: in general DKL(P||Q) ≠ DKL(Q||P), so KL is not a distance. If some Q(x) = 0 while P(x) > 0, that direction is ∞ (P puts mass where Q says it is impossible).
