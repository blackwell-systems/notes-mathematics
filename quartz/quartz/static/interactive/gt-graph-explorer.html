BFS uses a first-in-first-out queue, so it fans out level by level (shortest hop count from the start in an unweighted graph). DFS uses a last-in-first-out stack, so it plunges down one branch before backtracking. Both visit each vertex once; the order differs. Neighbors are explored in ascending label order here so the walks are reproducible.
