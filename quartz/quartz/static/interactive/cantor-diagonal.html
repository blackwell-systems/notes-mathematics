Cantor's diagonal argument

Any list of reals in [0,1] is incomplete: the diagonal always builds a number that is missing.

8

Click any digit to edit it (0-9). The diagonal is amber; the constructed number x updates live.

Construction rule. For each row i, take its i-th (diagonal) digit d and set x's i-th digit to d+1 if d<5, else d−1. This guarantees x's i-th digit ≠ ri's i-th digit, and it only ever produces digits 1–8, never 0 or 9 — so x has a single, unambiguous decimal form and we sidestep the 0.4999… = 0.5000… representation subtlety.
