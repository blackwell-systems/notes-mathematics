Definitions (bits)

H(X) = −∑x p(x) log2 p(x),   H(X,Y) = −∑x,y p(x,y) log2 p(x,y), with the convention 0 log 0 = 0.

I(X; Y) = H(X) + H(Y) − H(X, Y) = H(X) − H(X | Y) ≥ 0, with equality iff p(x, y) = p(x) p(y) for all x, y (independence).
